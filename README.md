# Portfolio Officiel - Ndeye Sermy Mergane

## Description

Ce projet est le portfolio officiel de Ndeye Sermy Mergane, une Data Engineer et Développeuse Web passionnée. Il présente ses compétences, projets et expériences professionnelles dans un design moderne et interactif.

## Technologies Utilisées

- **HTML5** : Structure du site web
- **CSS3** : Styles et animations personnalisées
- **JavaScript** : Interactivité (cursor personnalisé, etc.)
- **Google Fonts** : Polices Syne, DM Mono et Crimson Pro
- **SVG** : Effets visuels (noise overlay)

## Fonctionnalités

- Design responsive et moderne
- Curseur personnalisé interactif
- Navigation fluide avec scroll smooth
- Overlay de bruit pour un effet visuel unique
- Palette de couleurs sombre avec accents verts et violets
